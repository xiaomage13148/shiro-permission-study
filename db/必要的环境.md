## docker安装mysql

```bash
docker run -d -p 3306:3306 --privileged=true \
-v /home/mydata/mysql/conf:/var/log/mysql \
-v /home/mydata/mysql/data:/var/lib/mysql \
-v /home/mydata/mysql/conf:/etc/mysql \
-v /home/mydata/mysql/mysql-files:/var/lib/mysql-files \
-e MYSQL_ROOT_PASSWORD=wo610783 \
--name mysql8.0-3306 mysql:8.0 \
```
